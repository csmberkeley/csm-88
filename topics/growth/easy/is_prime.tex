\question Previously, we looked at the \texttt{is\_prime} function. Here's the code for it:

\begin{blocksection}
\begin{lstlisting}
def is_prime(n):
    if n == 1:
        return False
    k = 2
    while k < n:
        if n % k == 0:
            return False
        k += 1
    return True
\end{lstlisting}
\end{blocksection}
