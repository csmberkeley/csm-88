\question
\begin{lstlisting}
def fib_recursive(n):
   if n == 0 or n == 1:
       return n
   else:
       return fib_recursive(n - 1) + fib_recursive(n - 2)
\end{lstlisting}
\begin{solution}[0.0in]
   $\growth(\Phi^n)$, where $\Phi$ is the golden ratio. As long as you
   understand the runtime is exponential in n, we would accept your answer.\\
